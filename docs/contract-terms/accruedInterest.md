## Accrued Interest

**Acronym**: `IPAC`  
**Identifier**: `accruedInterest`  
**Group**: `Interest`  
**Type**: `Real`  
**Default**: ``  

### Description
Accrued interest as per SD. In case of NULL, this value will be recalculated using IPANX, IPCL and IPNR information. Can be used to represent irregular next IP payments.
