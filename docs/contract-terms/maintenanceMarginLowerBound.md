## Maintenance Margin Lower Bound

**Acronym**: `MRMML`  
**Identifier**: `maintenanceMarginLowerBound`  
**Group**: `Margining`  
**Type**: `Real`  
**Default**: ``  

### Description
Defines the lower bound of the Maintenance Margin. If MRVM falls below MRMML, then capital must be added to reach the original MRIM.

### Allowed Values
- `Positive`
