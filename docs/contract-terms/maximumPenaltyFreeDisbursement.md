## Maximum Penalty Free Disbursement

**Acronym**: `MPFD`  
**Identifier**: `maximumPenaltyFreeDisbursement`  
**Group**: `Optionality`  
**Type**: `Real`  
**Default**: `[ the value of notionalPrincipal ]`  

### Description
Defines the notional amount which can be withdrawn before XDN without penalty

### Allowed Values
- `Positive`
