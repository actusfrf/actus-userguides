## Fee Rate

**Acronym**: `FER`  
**Identifier**: `feeRate`  
**Group**: `Fees`  
**Type**: `Real`  
**Default**: ``  

### Description
Rate of Fee which is a percentage of the underlying or FER is an absolute amount. For all contracts where FEB does not apply (cf. business rules), FER is interpreted as an absolute amount.
