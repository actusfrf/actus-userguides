## Period Cap

**Acronym**: `RRPC`  
**Identifier**: `periodCap`  
**Group**: `Rate Reset`  
**Type**: `Real`  
**Default**: ``  

### Description
For variable rate basic CTs this represents the maximum positive rate change per rate reset cycle.

### Allowed Values
- `Positive`
