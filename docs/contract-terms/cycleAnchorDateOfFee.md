## Cycle Anchor Date Of Fee

**Acronym**: `FEANX`  
**Identifier**: `cycleAnchorDateOfFee`  
**Group**: `Fees`  
**Type**: `Timestamp`  
**Default**: ``  

### Description
Date from which the fee payment date schedule is calculated according to the cycle length. The first fee payment event takes place on this anchor.

### Allowed Values
- `ISO8601 Datetime`
