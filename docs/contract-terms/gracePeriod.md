## Grace Period

**Acronym**: `GRP`  
**Identifier**: `gracePeriod`  
**Group**: `Counterparty`  
**Type**: `Period`  
**Default**: `P0D`  

### Description
If real payment happens after scheduled payment date plus GRP, then the payment is in delay.

### Allowed Values
- `ISO8601 Duration`
