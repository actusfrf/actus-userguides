## Period Floor

**Acronym**: `RRPF`  
**Identifier**: `periodFloor`  
**Group**: `Rate Reset`  
**Type**: `Real`  
**Default**: ``  

### Description
For variable rate basic CTs this represents the maximum negative rate change per rate reset cycle.

### Allowed Values
- `Positive`
