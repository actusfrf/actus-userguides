## Non Performing Date

**Acronym**: `NPD`  
**Identifier**: `nonPerformingDate`  
**Group**: `Counterparty`  
**Type**: `Timestamp`  
**Default**: ``  

### Description
The date of the (uncovered) payment event responsible for the current value of the Contract Performance attribute.

### Allowed Values
- `ISO8601 Datetime`
