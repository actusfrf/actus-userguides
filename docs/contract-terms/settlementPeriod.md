## Settlement Period

**Acronym**: `STP`  
**Identifier**: `settlementPeriod`  
**Group**: `Settlement`  
**Type**: `Period`  
**Default**: `P0D`  

### Description
Defines the period from fixing of a contingent event/obligation (Exercise Date) to settlement of the obligation.


### Allowed Values
- `ISO8601 Duration`
